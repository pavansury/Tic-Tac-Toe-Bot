## 🎓 Code Example: Using the Agent

```python
from q_learning_agent import QLearningAgent
from game_engine import TicTacToeGame, GameTrainer

# Create agent
agent = QLearningAgent(learning_rate=0.1)

# Train
trainer = GameTrainer(agent)
trainer.train(episodes=1000)

# Play
game = TicTacToeGame(agent)
move = agent.select_action(game.board, training=False)
game.make_move(move, "O")

# Save
agent.save_model("my_model.pkl")
```
